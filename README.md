## Description
A `.jack` program compiler create with assembly language.

## Local Development
How to run locally:

### Compiling

Detect your system architecture:
- **Linux**: `uname -m`

> Currently this project only supports 64-bit architecture

Run `compile-x86_64.sh` file:
- **Linux**: `bash compile-x86_64.sh`
